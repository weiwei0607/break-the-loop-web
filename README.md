## 🎯 為什麼做這個？

現代人的一天充滿了「無意識行為」：
- 起床 → 滑手機 → 刷牙 → 滑手機 → 出門（全程沒意識到自己在做什麼）
- 坐在電腦前 → 3 小時後發現自己一直在刷新社群
- 想專心讀書 → 不知不覺打開 Netflix

這些都是 **「行為循環（Behavior Loop）」** — 大腦為了省電，把常用行為自動化，讓你不需要思考就能執行。

**Break the Loop 的解法**：
- 每天隨機抽一個「身體覺察」小任務
- 任務小到不可能失敗（「用非慣用手刷牙」）
- 目的是**打斷自動導航**，讓大腦重新上線

## 🏗️ 挑戰任務庫

任務分為三個難度，每個都設計成「身體覺察」導向：

### 🟢 簡單（2 分鐘內）
- 喝一杯溫水並深呼吸三次
- 用非慣用手刷牙
- 對鏡子裡的自己微笑五秒
- 站起來拉伸身體兩分鐘
- 閉目養神三分鐘
- 深蹲五次

### 🟡 中等（5 分鐘內）
- 赤腳在房間走一圈，感受地板溫度
- 閉眼吃一口食物，專注味道和口感
- 用嘴巴吹氣讓紙片移動 30 公分
- 單腳站立 30 秒
- 寫下此刻身體的五個感受

### 🔴 困難（10 分鐘內）
- 閉眼從房間一端走到另一端
- 用非慣用手寫自己的名字 10 次
- 用冷水洗臉後不擦乾，感受蒸發
- 躺地板上 5 分鐘，什麼都不做
- 對著鏡子描述自己的五官（不帶評價）

## 🧠 設計理念

這個 App 背後的設計原則來自 **正念（Mindfulness）** 和 **習慣中斷（Habit Interruption）**：

> 「打破自動化行為的最小有效劑量，不是改變整個習慣，而是**在循環中插入一個『意識時刻』**。」

每次完成一個挑戰，就是在告訴大腦：「嘿，我現在是有意識的。」累積這些小時刻，就能慢慢拿回生活的主控權。
